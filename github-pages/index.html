常见问题

文件存储在哪里？

文件存储在 GitHub Releases，这是 GitHub 官方的文件分发功能

有存储限制吗？

单个文件最大约 100MB（GitHub API 限制），仓库总大小建议不超过 1GB

安全吗？

完全安全。Token 只保存在你的浏览器本地，直接调用 GitHub API，不经过任何中间服务器

可以使用私有仓库吗？

可以，你的 Token 具有完整的仓库访问权限
