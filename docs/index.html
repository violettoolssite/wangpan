重要提示：

  • Token 必须有完整的 repo 权限
  • 文件将直接存储在 Git 仓库的 cloud-disk 目录
  • 支持最大 100MB 单文件
  • Token 只保存在浏览器本地，非常安全

遇到问题？

  • 403 Forbidden: Token 权限不足，请重新创建并勾选 repo
  • 404 Not Found: 检查仓库名和用户名是否正确
  • 401 Unauthorized: Token 无效，请重新生成

使用指南

1

创建 GitHub Token（重要）

访问 GitHub Settings → Tokens
必须勾选 repo 权限

2

创建存储仓库

在 GitHub 创建一个新仓库： https://github.com/new
可以是 public 或 private，记录下仓库名

3

连接并使用

填写 Token 和仓库信息，点击"连接仓库"，开始上传文件

常见问题

文件存储在哪里？

文件存储在 Git 仓库的 cloud-disk 目录，与其他代码一起管理

有存储限制吗？

单个文件最大 100MB，仓库总大小建议不超过 1GB

安全吗？

完全安全。Token 只保存在你的浏览器本地，直接调用 GitHub API

可以使用私有仓库吗？

可以，你的 Token 需要有访问该私有仓库的权限
